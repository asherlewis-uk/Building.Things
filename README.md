# Building.Things

A high-fidelity local IDE workspace shell with dual-mode interaction, SQLite-backed persistence, offline previewing, and a fully stubbed local AI layer.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up your environment variables if needed. Copy `.env.example` to `.env.local`:
    ```bash
    cp .env.example .env.local
    ```
4.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

- **Dual-Mode Interaction**: Write vs Chat modes.
- **SQLite Persistence**: Sessions, messages, files, artifacts, and deployment records are stored locally.
- **Offline Previewing**: Code and preview panels render saved workspace state without external services.
- **Virtual Terminal**: A predictable local pseudo-shell for navigating saved files.
- **Local Stubbed AI**: Chat responses are generated by `lib/ai-stub.ts` with no external AI API calls or API keys.
